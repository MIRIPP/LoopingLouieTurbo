# LoopingLouieTurbo
Speed up your LoopingLouie to make the game more fun


# Hardware

    - Arduino (any type works
    - LoopingLouie - the game
    - 9V Batterie
    - Pull Down Resistor 10 KOhm
    - MOSFET STP55NF06L
